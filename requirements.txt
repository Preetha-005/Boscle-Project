# ============================================
# Installation Notes
# ============================================

# 1. FFmpeg is required - install separately:
#    - Windows: Download from https://ffmpeg.org/download.html or use: choco install ffmpeg
#    - Linux: sudo apt install ffmpeg (Debian/Ubuntu) or sudo dnf install ffmpeg (Fedora)
#    - macOS: brew install ffmpeg
#
# 2. For GPU acceleration with Whisper, install appropriate PyTorch version:
#    - See: https://pytorch.org/get-started/locally/
#
# 3. Tkinter is usually included with Python, but on Linux you may need:
#    - Debian/Ubuntu: sudo apt install python3-tk
#    - Fedora: sudo dnf install python3-tkinter
#
# 4. API keys need to be configured in config/settings.json for:
#    - OpenAI (transcription and summaries)
#    - Google Cloud Speech (optional)
#    - Azure Speech Services (optional)
